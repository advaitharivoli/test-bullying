### 2. Multi-Level Authority Workflow
The system separates responsibilities across 4 distinct professional dashboards:
- **Teacher/Coordinator**: Initial review and triage.
- **Senior Mistress**: Focused oversight on escalated cases.
- **Head Mistress**: Investigatory level with inquiry controls.
- **Principal**: Final executive decision-making and school-wide stats.

### 3. Smart Escalation System
- **Real-Time Sync**: Descriptions and student data sync across all authority levels instantly.
- **Role-Based Views**: Each authority has a customized interface with specific action buttons (Solve, Take Over, Escalate).
- **CCTV Integration**: Simulated CCTV feed linking based on incident location.
